**Mob Controller**

> Mob controller handles all the mobs server-sided

```lua

-- Define mob controller 
local mob = require(path...)

-- Constructor
mob.new( type ) --> Returns the mob

-- Methods
mob:Spawn()     --> Spawns the mob at the start of the map
```
